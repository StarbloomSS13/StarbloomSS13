/tg/ Station 13 - Modules - Types — Var Details - Proc Details

unique_examine
/datum/element/unique_examine

An element to add a unique examine to something based on some conditions.

Vars

descThe special description that is triggered when desc_requirements are met. Make sure you set the correct EXAMINE_CHECK!
desc_requirementThe requirement setting for special descriptions. See examine_defines.dm for more info.
detective_sees_allIf TRUE, those with the detective's skillchip (TRAIT_SEE_ALL_DESCRIPTIONS) will be able to see the description regardless.
requirementsEverything we may want to check based on an examine check. This can be a list of JOBS, FACTIONS, SKILL CHIPS, or TRAITS, or a bitflag
special_affiliationOptional replacement for antagonist names. Ex: Noting something due to your "Donk Co" affiliation, instead of "Syndicate" affiliation
what_are_weA generic title for what we are, generated for things which hint.

Procs

get_departmentFormat a department bitflag into a string.
get_formatted_factionFormats some of the more common faction names into a more accurate string.
get_identifierGets an "identifier" for items which get a hint.
get_note_messageGets what message is displayed to the examiner. Returns a string, or null if they aren't able to see the special description.
on_examineSignal proc for COMSIG_PARENT_EXAMINE to hint this thing may have more info. Does not guarantee the examiner is someone who can actually access said info.
on_examine_moreSignal proc for COMSIG_PARENT_EXAMINE_MORE, shows the unique description of the examiner fits the bill.
